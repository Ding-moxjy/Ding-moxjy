去见海文哥哥

Germany — 跨越半个地球的兄弟之约

// 为什么想去

哥哥海文去德国已经好几年了。刚开始我还小，不太懂什么叫"远"，只记得他走的那天，在机场跟我挥手说"等长大点来看我"。那句话我一直记到现在。

每次视频，他都给我看那边的街道——干净的石板路、老旧的教堂、路边卖面包的小摊。他说周末会骑车去莱茵河边，秋天的时候满地落叶。那些画面让我觉得，哥哥生活的世界离我好远，又好像很近。

我不只是想去看风景，更想亲眼看看哥哥每天走过的路、吃过的东西、看过的天空。想和他一起走在德国的街头，像小时候那样，他在前面走，我在后面追。海文哥，等我，我一定会来的。

// 旅行计划
  1. 先搞定中考 📚
    这是第一关。把中考考好，才有底气跟爸妈说"让我去德国看哥哥吧"。
  2. 办护照和签证 🪪
    让哥哥帮忙确认签证材料，提前把护照办好，不耽误出发。
  3. 飞到德国找哥哥 ✈️
    漫长的飞行之后，落地就能看到哥哥在出口等我。想想就激动。
  4. 逛哥哥生活的城市 🏙️
    走一走他每天上下学的路，看看他常去的面包店和公园，感受他的日常。
  5. 一起去莱茵河边 🚶
    像哥哥说的那样，骑车去莱茵河边，秋天的话满地落叶，一定很美。
  6. 看一场球赛 ⚽
    德国足球氛围超棒，和哥哥一起在现场看一场球赛，那种感觉肯定一辈子忘不了。
  7. 拍照带回来 📸
    把这趟旅程的每一刻都记录下来，回来慢慢翻，也分享给朋友们。
